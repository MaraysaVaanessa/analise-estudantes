# Análise de IA nos Estudos

Esse projeto eu fiz para entender como o uso de inteligência artificial está influenciando os estudantes, principalmente depois da volta ao ensino presencial.

Eu analisei coisas como:

- frequência de uso da IA
- impacto na rotina
- dificuldades enfrentadas

Usei Power BI para montar os gráficos e visualizar melhor os dados.

Esse projeto foi importante pra mim porque consegui aplicar na pratica o que eu aprendi.
